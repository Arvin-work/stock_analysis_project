# 机器学习实际应用（股票预测）

## 程序目的
该程序是希望利用股市数据来完成机器学习，同时也借此完成对于python中大型工程文件的学习。以此了解应该如何完成大型工程文件的设计、构建和具体落实。

当然，**最主要**的，是为了接下来构架一个通用的数据分析代码。以此完成对于绝大多项目的快速落实。

## 程序流程设计
本程序分成以下几个层次。
- 项目配置层config_project
- 数据管理层data
- 引擎层engines
- 分析层analysis
- 展示层presentation
- 主程序main
- 工具集utils
