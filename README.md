# 机器学习实际应用（股票预测）

## 程序目的
该程序是希望利用股市数据来完成机器学习，同时也借此完成对于python中大型工程文件的学习。以此了解应该如何完成大型工程文件的设计、构建和具体落实。

当然，**最主要**的，是为了接下来构架一个通用的数据分析代码。以此完成对于绝大多项目的快速落实。

## 程序流程设计
本程序分成以下几个层次。
- 项目配置层config_project
- 数据管理层data
- 引擎层engines
- 分析层analysis
- 展示层presentation
- 主程序main
- 工具集utils

### 数据管理层
1. 模块目标
    
    该模块将为整个项目提供一个统一的数据存贮和调用的接口。该项目中数据的内外部访问（外部数据获取、数据读取和存储）均通过该模块完成。该模块需要完成以下工作，
    
    - 完成数据的自动管理（按照规定存储数据）
    - 完成数据的自动获取
    - 完成对于需求数据是否在本地数据的核实
    - 完成对于本地数据的查找和提供（暂时先不进行数据的本地提供，我们发现直接使用正常表达式访问即可）
